造飞机的工厂

午睡后醒来，睁开眼，还是白花花的电脑屏幕。刚才我似乎做梦了，努力的回忆，还是忘记了。
许多许多的梦，如同深海中的沉船，潜在记忆的最底部，淹没在时间的尘沙中。
我用“正午盘子”的订餐单折了一只纸飞机。
纸飞机，对于我不仅仅意味着童年。许多孤寂的时光里，我都会折纸飞机：一个人凝视窗外雨帘的时候，一个人凝视热腾腾的咖啡杯的时候，一个人在屋顶望向星空的时候，一个人坐在操场看台的时候，一个人……
在海边的时候，我折了很多的纸飞机。站在防波堤上，顺着海风的方向，掷向内陆的田野，有时候会滑翔得很远很远。偶尔我会放一只蚂蚁在上面，然后让风带着它，飞去一个迥然不同的天地。
我想站在北京最高的山上，向着四面八方放飞我折的飞机。

Jul 18, 2008
