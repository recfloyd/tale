翻校友录，2001年，我写的《断章》

　　９点５分，晨。 　　早晨的阳光穿越城市林立的高楼，射进窗子，在我枕边摩挲。睁开眼之后的第一件事，我开始 努力回忆昨晚所做的与你有关的梦。但是直到早饭过后，我还是没能记起梦境的细节。我经常这 样，在深夜里不知不觉卷入一个与你有关的虚幻的悲剧中。 　　我在雪地上，茫无目的地走着。积雪反射着白花花的阳光。大概是因为有点轻微雪盲的缘故， 我不由自主一个劲儿地揩眼泪。 　　此刻你在返家的列车上。 　　我努力睁着眼睛，翕动着鼻翼，搜寻着你的气息，你的足迹。想到我正是凭借这种方法，飞越 千里，抵至这个你客居的城市。白皑皑的积雪之上，是否每一个爱情故事都会有洁白美丽的结局 呢，我如此暗自思忖着。 　　２点１５分，午。 　　这样一个短暂无聊的假期，大部分人都拖着行李回家了。我还是不太适应这里的寒冷气候，但 是除了一个人在校园里游荡，我似乎无事可做。东风掠过雪原的时候，我也开始想念故园了。不知 道那些与我心手相牵的树木，此刻是否挺立着枯竭的树干，在这个冬季里绽放一种伟岸。 　　你曾经做过这么一个比喻，把我比作一棵树。寒冷的冬天，我总是有一点萧索和萎靡。 　　“你不是很喜欢绿色吗？”那个时候你转过头来，闪着一双澄亮的眼睛不解地问。是的，我不 是很喜欢绿色么？我花了很多时间，靠一种“属灵”感受那些树木的呼吸吐纳，我甚至以为我前世 就是一棵树。然而……我知道那并非我最终的归宿，我只是一棵无根的树。 　　松枝上的残雪，被风吹起，散落在空中，在阳光的照射下化作一道瞬间的虹，如此晶莹，仿佛 你流离在这个城市的眼泪，为你自己，也为我不羁的宿命。 　　１０点２０分，夜。 　　灯火辉煌的楼群中，悠扬地飘来一首歌，《悬崖》。我开始羡慕齐秦式的神话，靠一次圣域的 朝拜就从悬崖上挽回一段经典之爱。然而，我们呢？…… 　　你为什么要离开我呢，既然我们都暗恋了对方多年？ 　　你为什么又从废墟里把我捡回来呢，既然你已决定要放弃我？ 　　生命在我脚下延伸，我一直背着厚重的行囊，独步东西。我以为我可以依靠某种信仰生存，于 是我开始信仰你。后来我才知道，我必须也只能依靠自己。我必须成就自己，我必须在某种竞争中 赢得你。 　　在我这样的年龄里，很多人都犯过愚蠢的错。幸而我有机会得以弥补。在某个星火灿烂的夏 夜，你终于抱着我说爱我的时候，我今生第一次有了长远的目标，我要为你和我的明天担负爱的责 任。 　　１１点５分，夜。 　　没什么，我只是想说， 　　晚安…… 　　还有， 　　我爱你。
