滴汗，忘却的纪念

炎热的夏季，我泡了绿茶、菊花茶、茉莉花茶、苦丁茶……各种各样。灌下了这许多的茶之后，默默的坐着，静静的滴汗。 若干年之前在长春的夏夜，托着下巴在窗前听收音机，时任东北亚音乐台当家DJ的岳桐彤，以其舒缓而富有磁性的声音，介绍了张国荣的这一首《滴汗》，让我听得如痴如醉。

光阴荏苒，五年，十年，如飞而去。 在无数的夏夜里，我如此静静的滴下汗来。 我被流放于无人的荒野，游荡在无人的城市一隅。有时候会问自己，你在做什么，你在向哪里去？ 在某一年的某个夏夜，你滴着汗，把你的心和你的爱，永远留在了某个城市。 岳桐彤说，来吧，让我们一起滴汗。

Jul 23, 2008
