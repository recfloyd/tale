一无所有

假期前的最后一天，我上班迟到了，因为早晨准备出门的时候，遇到了一只流浪猫。 猫轻轻的挠在我的房门上，发出吱吱的声音。我打开门的时候，它轻悄悄的走了进来。这个不速之客，顿时让我陷入无比尴尬的境地，我挎着背包，左右为难，不知该如何是好。 猫并不可爱，莫若说因为瘦骨嶙峋而显得有些落魄。它笔直的走进床下，大概以为那里有什么美味吧，或者仅仅是想取暖也说不定。 床下一阵窸窸窣窣的声响，我蹲下来，静静的等它出来。过了一会，它安静的走过来，走到我面前。 我和猫对视了不到2分钟的时间，互相揣摩对方的用意。 听到猫的肚子在叫，类似于我饿的时候胃部发出的那种声响。我想，也许它需要些吃的东西。 翻箱倒柜…… 这样折腾了一番之后，我找到了两包饼干，两个苹果，三块奶糖，除此之外，再也没有能吃的东西。 心怀无限的愧疚，我摊开双手，示意我一无所有。猫大概未能顺利领会我的暗示，更近了一些倚在我腿间，伸出小小的凉凉的舌头舔在我的手掌上。 猫依偎在我身边的时候，我忽然很感动。深秋的早晨，一阵淡淡的暖意在心头升起。 猫的肚子发出咕噜噜的叫声。 我的肚子也发出咕噜噜的叫声。 我们依旧一无所有。

2008-09-28
