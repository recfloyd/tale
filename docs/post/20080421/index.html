致小胖子

列车在夜雨中坚决的挺进，我把脸贴在车窗上，笑了。我比你预计的要彪悍！

在归程的1/6处，我已经想明白了事情的前因后果，彻底的明白。于是，剩下的5/6，我无比释然。

我犹疑的原因在于，我没有自己的生活方式！我只是向往着、迎合着、模仿着别人的生活方式，而我自己，既没有固定不变的原则，也没有可供支撑的意志。像水一样，我流向不同的容器里，你们是什么模样，我就变成什么模样。

某人的名言——没有什么好与不好。

没有自己的生活方式，没有自己的性格，诚然会有不便的地方，但是你看，我花了1/6的时间，做了你们也许会做不到的事情——去试着理解别人的生活方式并迎合它。

我把问题简化了，因为在车厢尽头我悄无声息的运行了一个很有意思的算法——让一切变得简单。

小胖子，有件事情你以及其他的胖子们要知道，其实我很强大的！
