宿梦陛下翻了微臣的牌子

被宿梦翻了臣的牌子，无奈，只好奏对如下：

1.现在最想做的事情是什么？

  游遍北京所有的公园，拍2万张照片

2.近阶段烦恼的事是什么？

  孤单的时候想找人说话，找不到

3.如果过年回家，有个家伙硬要去火车站（机场）接你，你会答应不？

  会吧，我看看他到底出于什么目的

4.如果你有资金开店当老板，你会想开什么店？

  咖啡馆

5.你最尴尬的事是什么？

  误入女WC且同班同学正在里面

6.你喜欢一个人还是大家聚在一起？

  我喜欢热闹

7.你最想去哪个地方？为什么？

  深圳，有山有海的地方

8.最常有的想法是什么?

  今天晚饭怎么解决，每天15：30点准时思考该问题

9.记得好朋友的生日吗?

  记得

10.对你来说最大的幸福是什么？

   寻找到生命意义，我是认真的

11.一直以来最大的心愿?

   走遍世界每个角落

12.受伤的人是不是都有恋爱恐惧症？

   总结教训重新开始

13.最想和你喜欢的人在一起做什么事情?

   两个人吃甜点

14.你会在乎你爱的人（现在是你的女、男朋友）心里爱着别人吗？

   宿梦，这是你提的问题？这不明摆着吗，不在乎的话也太2了吧

15.忠于感情还是忠于责任？

   感情

16.你最想同点你名的人说的一句话？

   宿梦陛下万岁万岁万万岁

17.你是否有蓝颜知己（如果你是女生），或红颜知己（如果你是男生）？你觉得会一直保持这种关系吗？

   有，不知道，但这玩意儿很悬就像核技术，搞好了就是发电厂造福人类，搞不好就是原子弹

18.你会选择一个你很爱的人，还是很爱你的人过一辈子？

   我选后者，前者试过了，失败告终

19.你的目标是？

   体会到人生的意义，同10，我认真的

20.遇到不顺心，不开心的时候，你怎么开解自己？

   步行、逛商店、吃甜点，然后关闭当前进程，restart

我的替换：

第9题我觉得没啥意思，改为：你认为自己的命运是可以由自己掌控的吗？

我要翻牌子了：极品小伙，小李（小李还在闭关中吗？），了一里，提拉米苏，幸福的颜色

规则：被点到的名字要在自己的空间里写下自己的答案，然后去掉一个你最不喜欢的问题再加上一个你的问题，仍然组成20个问题，传给5个人，列出其他需要回 答问题的人的名字，还要到这5个人的空间里留言通知对方－－你被点名了，被点名者不得拒绝回答问题，完成游戏的人将会得到大家的祝福
