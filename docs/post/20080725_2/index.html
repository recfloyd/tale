一生的谎言

转身，笑靥，窗前。 你喃喃低语， 我的梦在焰火中暝灭。 你淡定从容， 闪烁的只有黑暗中你的眼。 时间的断片， 散落一地。 映照着我卑微的人生， 映照着你的谎言。 你是一本难解的书， 跌宕，华彩。 你自己撰写的片段。 一页页的翻， 从头至尾， 你却没有为我留下任何一字， 任何一个标点。 看到结尾处我笑了， 咯咯的笑。 我喜欢美丽的谎言。 如果可以， 我愿意用一生兑换。 只是…… 并非你的谎言不够圆满， 而是我还不习惯自我欺骗。 我想你帮我。 在我醒来之前， 不要从梦中叫醒我。 我喜欢你的谎言， 我喜欢你说那个词， 永远。

2008-07-25
